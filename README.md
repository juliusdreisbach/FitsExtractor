# FitsExtractor
FITS Extractor is a small python GUI application that allows 1D spectrum extraction from FITS file binary tables

# How to start the application
Run the command
fits_extractor.py
to execute the script. The graphical interface will open automatically. The script requires some non-standard python packages to be installed. To install them, run the command
pip install -r requirements.txt
in the according directory or install the required packages by yourself. Check the provided “requirements.txt” file for information about the packages you need to install.
